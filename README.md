# Sistema Éxito

## Descripción

Sistema Éxito es una aplicación de escritorio desarrollada en Java para la gestión comercial de una tienda. Permite administrar categorías, clientes, proveedores, productos y ventas mediante una interfaz gráfica desarrollada con Java Swing y una base de datos MySQL.

## Características

- Inicio de sesión
- Gestión de categorías
- Gestión de clientes
- Gestión de proveedores
- Gestión de productos
- Registro de ventas
- Conexión a base de datos MySQL

## Tecnologías utilizadas

- Java
- Java Swing
- JDBC
- MySQL
- Maven
- IntelliJ IDEA
- Git
- GitHub

## Cómo ejecutar el proyecto

1. Instalar Java JDK.
2. Instalar MySQL.
3. Crear la base de datos `exito`.
4. Importar las tablas.
5. Abrir el proyecto en IntelliJ IDEA.
6. Ejecutar la clase `Main.java`.
